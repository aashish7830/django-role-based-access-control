# Django Role-Based Access Control System

## 📌 Project Overview
This project is a Role-Based Access Control (RBAC) system built using Django.

It allows multiple users to manage subject-specific data with restricted access.

## 🚀 Features
- Multi-user system
- Role-based access using Django Groups
- Subject-wise filtering (Hindi, English, Math, Science)
- Secure admin panel
- Custom queryset filtering

## 🛠 Tech Stack
- Python
- Django
- MySQL

## ⚙️ Setup Instructions
1. Clone repo
2. Run:
   python manage.py migrate
   python manage.py runserver
